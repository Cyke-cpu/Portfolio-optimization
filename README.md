# Modern Portfolio Optimizer
An interactive Python finance app thats uses Modern Portfolio Theory to optimize a portfolio of stocks and ETFs

# Tools used

- Python 
- Streamlit
- Yahoo Finance
- Pandas
- NumPy
- SciPy
- Matplot
- Seaborn

## Default Assets
AAPL, MSFT, NVDA, JPM, XOM, GLD, TLT, SPY

## Project Purpose

I built this project to better understand potfolio construction, risk-return tradeoffs, and how quantitative methods are used in investment management
